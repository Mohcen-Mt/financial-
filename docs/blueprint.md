# **App Name**: Financial Alchemist | الخيميائي المالي

## Core Features:

- Authentication: Secure login/registration with username, email, password, and PIN quick lock.
- 3D Dashboard: Interactive dashboard displaying daily/weekly/monthly/total profit, product totals, sales metrics, and smart alerts using 3D charts.
- Product Management: Add, edit, and delete products with details like name, category, price, quantity, color, size, and image upload, including auto profit calculation.
- Sales Tracking: Record sales transactions, including product selection, quantity sold, customer details, and payment method, with auto profit calculation.
- Profit Analytics: Analyze profit trends with daily/weekly/monthly/yearly breakdowns, net profit calculations, and top product insights using 3D timeline charts.
- AI Pricing Tool: An AI-powered tool provides optimal selling price suggestions, analyze profit trends, warn about low-selling products, and suggest restocking items, and deliver daily smart tips (mock logic, no external API).
- Data Export: Export products, sales, expenses, and monthly profits as CSV, Excel, and PDF files.

## Style Guidelines:

- Background: Soft dark gray (#333333) to provide a sophisticated and calm canvas.
- Primary: Blue gradient (#6495ED to #4682B4) evoking trust and reliability.
- Accent: Light purple (#A020F0) to highlight key elements and calls to action.
- Body: 'PT Sans' (sans-serif) for readability and a modern feel.
- Headlines: 'Space Grotesk' (sans-serif) for a bold, modern and technical appeal. Use 'Inter' as a body font.
- Note: currently only Google Fonts are supported.
- Use SVG icons throughout the app for scalability and a clean, crisp look.
- Glassmorphism with soft 3D elements to give depth, smooth animations for micro-interactions, and a fully responsive design.
- Subtle, professional animations to enhance user experience without being distracting.
